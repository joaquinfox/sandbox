Welcome to my sandbox, a place to practice the skills needed to be a front-end developer.

This site will be part note-pad, recording my own observations and lessons learned, and part tutorial. The effort is inspired in part by the notion that teaching, is an aid to learning, and that giving back is part of building the open source community.

Becoming a Web-Developer

Setup a development environment

  • This will have two main parts;
    • A Unix terminal, also known as the Command Line, and...
    • A text editor.

Set up a Git account

  • (like GitHub or Bitbucket) and learn the basics of how it works and what it is used for, including,
    • what it means to save, commit, push and pull.
    • how to deploy a website.
    • creating branches and merging.

Third Step

  • Learn the basic html structure.
  • Learn CSS.
  • Practice, practice, practice, by building, building, building.

Next Steps

  • Learn Java Script.
  • Learn the Unix Command Line really well.
